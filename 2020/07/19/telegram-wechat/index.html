(使用 EWS 有 微信网页版被封的危险，请谨慎使用)

安装

建议先使用 # apt update && apt upgrade -y更新所有软件包

二进制依赖

# apt install -y python3-pip python3-wheel ffmpeg libmagic1 libwebp6

主体
